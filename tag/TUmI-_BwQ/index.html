通过configmap控制kubelet
· 2019-06-20 · # kubernetes # kubelet # configmap

简介

在 Kubernetes 1.8 版本上，除了可以通过命令行参数外，还可以通过保存在硬盘的配置文件设置 Kubelet 的配置子集。 将来，大部分现存的命令行参数都将被废弃，取而代之以配置文件的方式提供参数，以简化节点部署过程。 动态配置kubelet在1.11进入beta版本
